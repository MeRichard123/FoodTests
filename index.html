Food Tests

Testing samples of food for different molecules

Testing for Starch

  1. Add Iodine
  2. If the Colour Changes from yellow-orange to blue-black
  3. Then Starch is present

Testing for Lipids

Using the Ethanol emulsion test

  1. Mix with Ethanol
  2. SHAKE!!
  3. Pour Into Some Water
  4. SHAKE AGAIN!!
  5. Fats and oils with float to the top forming a cloudy emulsion

Testing for Protein

The Biuret Test

  1. Potassium Hydroxide is mixed with the food
  2. Two Drops of Copper Sulphate are added
  3. Potassium Hydroxide + Copper Sulphate = Biuret Solution
  4. Change from pale blue to purple is a positive test

Testing for sugars

Reducing Sugars Only Like Glucose

  1. Mix with Equal Volume of Benedict's Solution
  2. Place in a HOT Bath for a few minutes
  3. Reducing sugars cause a Reaction
  4. It May Form a precipitate

Benedics is light Blue by default
